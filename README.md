## 📝 과제 제출 방법

> 포크 방식이 아닌 **공동 레포지토리 협업 방식**으로 진행합니다.

**1. 공통 저장소 클론**
```bash
git clone https://github.com/sallydeveloperr/likelion14swu-DjangoStudy-Django-Cubs.git
```

**2. 본인 이름으로 브랜치 생성 및 이동**
```bash
git checkout -b [본인 브랜치명]
```

**3. 본인 이름 폴더로 이동 후 과제 코드 작성**

**4. 원격 브랜치에 업로드**
```bash
git add .
git commit -m "커밋 메시지"
git fetch origin
git pull origin main
git push origin [본인 브랜치명]
```

**5. GitHub에서 Pull Request 생성 → main 브랜치로 merge**

> 🔔 팀원당 최소 **PR 3회** 필수  
> ❓ 질문은 디스코드 `Backend - 질문 답변` 채널로!
